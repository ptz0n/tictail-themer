# Tictail Themer

This is a tool to help you build and test your Tictail theme in your local environment.

## Installation

1. Install [Node.js](http://nodejs.org/)
1. `$ npm install gulp -g`
1. Clone the repo
1. `$ npm install`
1. `$ gulp config`

## Deploying your theme
